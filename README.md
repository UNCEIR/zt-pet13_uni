# VibeCoding -- 宠物定位系统

一个功能完整的宠物管理和定位平台，集宠物档案、GPS定位追踪、健康管理、社区交互和电商服务于一体。基于uni-app开发，支持H5和微信小程序。

## 📱 项目信息

- **项目名称**：VibeCoding - 宠物定位系统
- **版本**：1.0.0
- **技术框架**：uni-app + Vue 2
- **支持平台**：H5、微信小程序
- **开发语言**：Vue 2、HTML、CSS

## ✨ 核心功能

### 🐾 宠物档案管理
- **宠物信息**：宠物列表查看、宠物ID卡、编辑宠物基本信息
- **宠物档案**：品种选择、性别选择、宠物健康记录
- **养主管理**：添加宠物主人、家庭成员管理、联系方式管理

### 📍 GPS定位与追踪
- **实时定位**：设备选择和绑定、实时GPS追踪
- **追踪历史**：历史轨迹查询、今日追踪记录
- **电子围栏**：围栏设置、围栏热点管理、进出通知提醒

### 💪 健康管理
- **健康记录**：宠物健康数据记录和展示
- **健康档案**：定期健康体检信息管理

### 🏘️ 发现页面（社区与电商）
- **社交互动**：
  - 动态发布与分享
  - 活动创建/详情展示
  - 话题讨论
  - 用户个人资料和关注

- **问答社区**：
  - 问题创建和详情
  - 知识库问答

- **电商服务**：
  - 产品展示和详情
  - 购物车管理
  - 订单确认和管理
  - 订单详情查询

### 💬 消息与沟通
- **消息中心**：接收通知、消息管理
- **邀请方式**：支持多种邀请渠道

### ⚙️ 用户设置
- **我的**：个人信息、偏好设置
- **语言设置**：多语言支持
- **系统版本**：版本检查和更新
- **使用指南**：应用手册和帮助文档

```
│   ├── mine/                       # 我的页面

├── images/                         # 应用图标/制表符图片

├── main.js                         # 应用入口
├── app.css                         # 全局样式
└── package.json                    # 项目配置
```

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 本地开发

#### H5平台
```bash
npm run dev:h5
```

#### 微信小程序
```bash
npm run build:mp-weixin
```

### 生产构建

#### 构建H5
```bash
npm run build:h5
```

#### 构建微信小程序
```bash
npm run build:mp-weixin
```

## 📦 核心依赖

- **Vue**: 2.6.14 - 前端框架
